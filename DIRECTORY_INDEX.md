# Directory Index

## Top-Level Folders

`agent/`
Application source code for enrichment, channel handlers, CRM integration, calendar integration, and orchestration.

`data/`
Committed input snapshots used by the enrichment layer, including Crunchbase ODM sample data and the layoffs.fyi CSV snapshot.

`eval/`
Evaluation harness, score logs, latency outputs, and benchmark trajectories for Act I and Act IV measurement.

`market/`
Supporting market analysis artifacts used to justify segment prioritization and market-space framing.

`method/`
Mechanism design documentation, ablation results, and held-out traces for the SCAP approach.

`planning/`
Execution planning artifacts, milestone sequencing, and dependency mapping for the challenge submission.

`probes/`
Adversarial probes, failure taxonomy, and target failure mode documentation used to motivate the mechanism work.

`qa/`
Submission-level validation notes and final readiness checks.

`report/`
Executive memo material, evidence graph assets, and utilities for claim validation.

`seed/`
Seed content and challenge input material used to ground messaging and scenario assumptions.

`tau2-bench/`
Local benchmark code and benchmark-generated outputs used for reproduction and evaluation.

`tenacious_sales_data (1)/`
Imported business context material and challenge-provided sales reference assets retained alongside the repo for static review.
